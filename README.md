# SR Tasks

A stylish task manager web app built with HTML, CSS, and JavaScript.

## Overview

`SR Tasks` helps users organize tasks with a polished hero screen, priority selection, due date support, and theme toggle.

## Files

- `index.html` - main web page structure
- `style.css` - visual styling and layout
- `script.js` - task interaction and behavior
- `server.js` - optional server file (if using local hosting)

## Live Demo

https://internpe-task-3-73l9.vercel.app/

## How to Run

1. Open `index.html` in your browser for a local preview.
2. Or run a local server and navigate to `http://localhost:PORT`.

## Features

- Add tasks with due dates
- Priority selection
- Animated hero screen
- Theme toggle
